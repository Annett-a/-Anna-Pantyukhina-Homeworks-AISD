## Работа с очередью (Queue)

Реализацию обобщённой очереди Queue<T> на основе односвязного списка.

### Файлы проекта

Queue.java

Класс Queue<T>:

enqueue(T data) — добавить элемент в хвост очереди;

dequeue() — удалить и вернуть элемент из головы очереди;

peek() — вернуть первый элемент без удаления;

isEmpty() — проверить, пуста ли очередь;

size() — получить количество элементов;

clear() — очистить очередь;

toString() — вывести содержимое в формате {elem1, elem2, ...}.

### Main.java

Консольное меню для взаимодействия с Queue<Integer>:

enqueue X — добавить число X в очередь

dequeue — удалить и вывести первый элемент

peek — вывести первый элемент без удаления

isEmpty — проверить пустоту

size — узнать размер очереди

clear — очистить очередь

print — вывести все элементы

exit — выйти из программы
