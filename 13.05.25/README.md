### PeakFinder — поиск пика в битоническом массиве

Консольная Java-программа, реализующая алгоритм поиска максимальной вершины (пика) в битоническом массиве за O(log n) с помощью бинарного поиска.


Файлы проекта

Main.java — содержит:

метод findPeak(int[] a) для поиска индекса пика;

main для ввода данных пользователем и вывода результата.


Алгоритм

Битонический массив — сначала строго возрастает, затем строго убывает. Метод findPeak работает так:

Инициализируем указатели low = 0, high = a.length - 1.

Пока low < high:

Вычисляем mid = low + (high - low) / 2.

Если a[mid] < a[mid + 1], значит пики правее, устанавливаем low = mid + 1.

Иначе пики слева или на самом mid, устанавливаем high = mid.

Возвращаем low — индекс пика.
