### Стек с поддержкой минимума за O(1)



Stack.java — базовый класс стека на массиве:

push(int x) — добавить элемент;

pop() — удалить и вернуть верх;

peek() — вернуть верх без удаления;

isEmpty(), isFull() — проверки состояния.


NewStack.java — расширенный стек с поддержкой минимума:

Внутри хранит два Stack:

stack1 для данных;

stack2 для отслеживания минимума на каждом уровне.


Методы:

push(int x) — помещает в оба стека, в stack2 сохраняет либо x, либо текущий минимум;

pop() — удаляет из обоих стеков;

peek() — возвращает stack1.peek();

getMin() — возвращает stack2.peek().

Main.java — демонстрация работы:

Инициализирует NewStack, выполняет несколько push/pop и выводит текущий минимум.


Описание реализации

Для поддержания минимума используется жадный дублирующий стек:

При добавлении x в stack1:

Если stack2 пуст или x <= stack2.peek(), пушим x в stack2;

Иначе пушим в stack2 значение stack2.peek() — тем самым на каждом уровне дублируем предыдущий минимум.

При удалении элемента синхронно делается pop() в обоих стеках.

getMin() просто возвращает верхний элемент stack2 — текущий минимум.

Таким образом, все операции (push, pop, peek, getMin) выполняются за O(1).
