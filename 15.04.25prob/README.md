## Работа с очередью на основе ArrayDeque

### Файл проекта

Main.javaКласс с методом main, в котором создаётся очередь через интерфейс java.util.Queue и класс java.util.ArrayDeque. Программа предоставляет текстовое меню для операций:

enqueue X (через offer) — добавить элемент X в конец очереди;

dequeue (через poll) — удалить и вывести первый элемент;

peek — вывести без удаления первый элемент (peek);

isEmpty — проверить, пуста ли очередь;

size — вывести количество элементов;

clear — очистить очередь;

print — вывести всё содержимое очереди;

exit — завершить работу программы.
